ps、date、wc命令

ps

​ Linux下会运行各种各样的进程，如果要进行监控我们必须对这些进程进行监控；ps命令可以看到进程的运行状态、进程是否结束、哪些进程占用了过多的资源等。但注意，ps命令是显示瞬间进程的状态，并不会动态连续跟进进程状态，如果想对进程进行实时监控应该用top命令。

date

​ date命令可以按照指定格式显示日期，只键入date则可以默认显示当前时间。

wc

​ 在Linux下wc命令用于计算字数，利用wc指令我们可以计算文件的Bytes数、字数、或是列数。

下面是三个命令对应参数的使用方法：

  • ps
  • date
  • wc

ps [选项]

选项 功能
-A 显示所有进程，跟-e效果一样
-a 显示现行终端机下的所有进程，包括其他用户的进程
-u 显示当前用户的进程状态
-x 通常与-a一起使用，方便列出完整的信息
-l 较长，较详细的将PID的信息列出
-j 工作的格式（jobs format）
-f 把进程的所有信息都显示出来
-e 表示所有继承

date ”+今天是%Y“（格式）

选项 功能
%n 下一行
%t 跳格
%H 小时（1-12）
%k 小时（0-23）
%l 小时（1-12）
%M 分钟（00-59）
%p 显示本地AM或PM
%r 直接显示（24小时制）
%S 秒
%A 显示星期几（Sunday格式）
%B 月份
%j 一年中的第几天
%Y 年
%m 月
%d 日
%x 直接显示日期

wc [-c -l -w] [-help] [–version] [文件]

选项 功能
-c（–chars） 只显示bytes数
-l（–lines） 只显示行数
-w（–words） 只显示字数
–help 在线帮助
–version 显示版本信息

cat、rm、echo命令

cat

​ 在终端设备上显示文件内容，cat命令只适合查看内容较少的文件。对于内容多的文件可以用more命令查看。

rm

​ 删除文件或目录，一次可以删除多个文件，或递归删除目录及其内的所有子文件。

echo

​ 在终端设备上输出指定字符串或变量提取后的值，可以将输出的指定字符串内容同管道符一起传递给后续命令作为标准输入信息进行二次处理，还可以和输出重定向符一起操作，将信息直接写入文件。

下面是三个命令对应参数的使用方法：

  • cat
  • rm
  • echo

cat [参数] [文件名]

选项 功能
-A 等价于-vET参数组合
-b 显示行数（空数不编号）
-e 等价于-vE组合
-E 每行结束处显示$符号
-n 显示行数（空行也编号）
-s 显示行数（多个空行显示一个编号）
-t 等价于-vT组合
-T 将TAB字符显示为^|符号
-v 使用^和M-引用
–help 显示帮助信息
–version 显示版本信息

rm [参数] [文件名]

选项 功能
-d 仅删除无子文件的空目录
-f 强制删除文件而不询问
-i 删除文件前询问用户是否确认
-r 递归删除目录及其内全部子文件
-v 显示执行过程的详细信息
–help 显示帮助信息
–version 显示版本信息

echo [参数] [字符串或$变量名]

选项 功能
-e “\a” 发出警告音
-e “\b” 删除前面一个字符
-e “\c” 结尾不加换行符
-e “\f” 换行后光标仍停留在原来的位置
-e “\n” 换行后光标移至行首
-e “\r” 光标移至行首但不换行
-E 禁止反斜杠转义
-n 不输出结尾的换行符
–version 显示版本信息
–help 显示帮助信息

ping、netstat、ssh命令

ping

​ 测试主机间网络的连通性，发送出基于ICMP传输协议的数据包，要求对方主机予以回复。

netstat

​ 显示各种网络相关信息，例如网络连接状态、路由表信息、接口状态、NAT、多播成员等。这个命令还可在windows所有系统上运行。

ssh

​ 安全的远程连接服务器主机系统。

下面是三个命令对应参数的使用方法：

  • ping
  • netstat
  • ssh

ping [参数] [域名或IP地址]

选项 功能
-4 基于IPv4协议
-6 基于IPv6协议
-a 发送数据包时发出蜂鸣声
-b 允许ping一个广播地址
-c 设置发送数据包的次数
-d 使用接口的SO_DEBUG功能
-f 使用泛洪模式大量向目标发送数据包
-h 显示帮助信息
-i 设置收发信息的间隔时间
-l 使用指定的网络接口送出数据包
-m 仅输出数值
-p 设置填满数据包的范本样式
-q 静默执行模式
-R 记录路由过程信息
-s 设置数据包的大小
-t 设置存活数值TTL的大小
-v 显示执行过程详细信息
-V 显示版本信息

netstat [参数]

选项 功能
-a 显示所有连接中的接口信息
-A 设置网络连接类型
-c 持续显示网络状态
-C 显示路由配置信息
-F 显示路由缓存信息
-g 显示多重广播功能群组成员名单
-h 显示帮助信息
-i 显示网络界面信息表单
-l 仅显示正在监听的服务状态
-n 直接使用IP地址而不是域名
-N 显示网络硬件外围设备的符号链接名称
-o 显示计时器数据信息
-p 显示正在使用接口的程序识别码和名称
-r 显示路由表信息
-s 显示网络工作信息统计表信息
-t 显示TCP传输协议的连线状态
-u 显示UDP传输协议的连线状态
-V 显示版本信息

ssh [参数] [域名或IP地址]

选项 功能
-1 使用SSH协议版本1
-2 使用SSH协议版本2
-4 基于IPv4协议
-6 基于IPv6协议
-a 关闭认证代理连接转发功能
-A 开启认证代理连接转发功能
-b 设置本地对外提供服务的IP地址
-c 设置会话的密码算法
-C 压缩所有数据
-f 后台执行ssh命令
-F 设置配置文件
-g 允许远程主机连接本机的转发端口
-i 设置密钥文件
-l 设置登录名
-N 不执行远程指令
-o 设置配置参数选项
-p 设置远程服务器上的端口号
-q 静默执行模式
-s 请求远程主机上的子系统调用
-v 显示执行过程详细信息
-V 显示版本信息
-x 关闭x11转发功能
-X 开启x11转发功能
-y 信任x11转发功能
